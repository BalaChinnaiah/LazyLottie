# Lazy Lottie 

## It is used to convert Lottie JSOn files into Transparent High Quality GIF's.

### How it works?

1. It will First Take the Json as Input and Convert them into transparent PNG Images and store it in a Frames Folder
2. Then After Frames generated, It will combine All Frames into a Sequence and make a GIF
3. And the command you run "node convert.js targetted_file.json output_filename.json" , the outputfile will be saved in the same Directory.
